### Use

1. 로그인 및 회원 가입<br>
2. 로그인을 하면 MyPage에서 나의 일기 모아보기 기능
3. 일기는 영화 검색(네이버 영화 검색 API), 제목과 부제목, 별점, 태그 , 포스터 , 일기 내용으로 이루어짐
4. 다른 유저의 일기 공유 <div>
